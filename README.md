## Usage

1. Clone the project to your computer:
   ```bash
   git clone https://github.com/0MeMo07/MeMo-Burger.git
   ```

2. Navigate to the project directory:
   ```bash
   cd MeMo-Burger
   ```
3. Install the required packages:
   ```bash
   npm ci
   ```
   or
   ```bash
   npm install <packages name>
   ```
4. Start the project::
   ```bash
   npm run dev
   ```
